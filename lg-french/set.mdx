## Présentation

La page **Sets & Minifigs** présente l'ensemble des informations utiles concernant un set ou une minifig. Elle est accessible en saisissant une référence de set ou de minifig (selon la nomemclature Bricklink&reg;) dans la `zone de recherche`. En fonction de vos [paramètres](/lg-french/settings) les informations et les widgets affichés peuvent varier.
