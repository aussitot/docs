## Bienvenue sur votre tableau de bord !
Cette page centrale est votre hub personnel, vous donnant une vue d'ensemble de vos collections et de vos transactions. Conçu pour simplifier votre expérience d'utilisateur, le tableau de bord rassemble toutes les informations essentielles.

Vous y trouverez un résumé de vos récentes interactions sur le site. Des statistiques claires et concises vous permettent de suivre vos actions et la valeur de vos collections en un coup d'œil.

Vous êtes prêt à commencer ? Plongez dès maintenant dans votre tableau de bord et découvrez la présentation des différents widgets qui le composent.

## Mise à jour

Si, dans la page des [paramètres](/lg-french/settings), vous avez sélectionné l'option `mise à jour automatique`, vous n'avez rien à faire. La page sera mise à jour dès que vous la consulterez.
Lors de l'affichage du tableau de bord, le système effectue de nombreux calculs et interroge les API pour récupérer les dernières données de prix et de disponibilité. **Lorsque votre collection devient importante, le temps de chargement de la page du tableau de bord peut devenir significatif**. Il est donc conseillé de désactiver l'option "mise à jour automatique". Vous devrez alors cliquer sur le bouton "Refresh" en haut à droite du tableau de bord pour le mettre à jour.

## Widgets
Les widgets composent votre tableau de bord. Composés de chiffres et de graphiques ils vous permettent d'avoir une vue synthétique sur votre collection. Retrouvez ci-dessous la description des principaux widgets constituant le tableau de bord.

<Tip>Les graphiques sont interactifs. Vous pouvez cliquer dessus pour voir les différentes valeurs. Pour les graphiques avec légende, cliquez sur celle-ci pour sélectionner les courbes à afficher.</Tip>
