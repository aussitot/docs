  <Accordion title="Il existe une application mobile ?" icon="mobile-notch">
    Oui et... non :)
    Vous ne trouverez pas sur l'AppStore ou Google play d'application Legaussitot.

    **Legaussitot est une PWA (Progressive Web App)**. 
    Une application PWA, ou Progressive Web App, est une application web qui utilise des technologies web modernes pour fournir une expérience utilisateur similaire à celle d'une application native, mais qui peut être accessible via un navigateur web. Les PWA combinent les meilleures caractéristiques des sites web et des applications mobiles, offrant une expérience rapide, fiable et engageante, tout en étant facilement installables et accessibles à partir de n'importe quel appareil ou plateforme.
    
    En résumé, les PWA offrent une alternative flexible et puissante aux applications mobiles traditionnelles, offrant une expérience utilisateur améliorée tout en simplifiant le processus de développement et de distribution pour les développeurs.
    ### Installer l'application sur mon écran d'accueil
    Sur un appareil <Icon icon="apple" color="000000"/> Apple, comme un iPhone ou un iPad, vous pouvez installer une Progressive Web App (PWA) en suivant ces étapes :
    1. Ouvrir le navigateur Safari: Assurez-vous que vous utilisez le navigateur Safari, car c'est le seul navigateur sur les appareils iOS qui prend en charge l'installation de PWA.
    2. Accéder au site web : https://www.legaussitot.com/admin.php

    3. En bas de la fenêtre Safari, cliquez sur le bouton de partage <Icon icon="arrow-up-from-square" iconType="solid" color="000000"/> puis choisissez **Sur l'écran d'accueil** <Icon icon="square-plus" iconType="regular" color="000000"/>

    Une fois ajoutée à l'écran d'accueil, l'icône de l'application apparaîtra comme n'importe quelle autre application. Vous pouvez maintenant l'ouvrir en appuyant sur son icône.
    La PWA fonctionnera comme une application native, même si elle est techniquement exécutée à partir du navigateur Safari.
    <Info>Pour les appareils sous <Icon icon="android" color="000000"/> Android la procédure est à peu prêt identique. La plupart des navigateurs modernes sur Android prennent en charge les PWA, notamment Google Chrome, Mozilla Firefox, et Microsoft Edge.</Info>
